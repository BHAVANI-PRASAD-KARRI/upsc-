# 🇮🇳 UPSC First Attempt Preparation Journey

> “Discipline beats motivation.”  
> My complete roadmap for cracking the UPSC Civil Services Examination in the very first attempt.

---

## 📌 About This Repository

This repository contains my preparation strategy, notes structure, study resources, timetable, and progress tracking for the UPSC CSE Examination.

The goal is to:

- Crack UPSC in the **first attempt**
- Build strong conceptual clarity
- Maintain consistency and discipline
- Cover Prelims + Mains + Interview systematically

# 🧭 UPSC Exam Structure

## Stage 1: Prelims
- GS Paper-I
- CSAT (Paper-II)

## Stage 2: Mains
- 9 Descriptive Papers

## Stage 3: Personality Test
- UPSC Interview
